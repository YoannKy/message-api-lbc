# message-api-lbc

## Introduction:
This is a REST API that allow:
  - To list messages
  - To create a message
  - To show a message
  - To delete a message

A message is composed of
- A public status (is the message private or public ?)
  - A text
  - See HTML in the right
  - Magic

## Installation
```sh
$ npm i
$ npm start
```
## Endpoints
 ### GET /messages
 ### POST /messages
 ### GET /message/:id
 ### PUT /messages/:id
 ### DELETE /messages
